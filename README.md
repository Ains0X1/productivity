# Student Productivity App

A modern and beginner-friendly productivity software built using HTML, Tailwind CSS, and JavaScript.

This application helps students manage daily tasks such as homework, assignments, study goals, and reminders through a simple and interactive interface.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Live task statistics
- Responsive modern UI
- Beginner-friendly code structure

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript

## How It Works

Users can enter tasks in the input field and add them to the task list. Each task can be completed using a checkbox or removed using the delete button. The app also displays the total number of tasks and completed tasks in real time.

## Learning Objectives

This project helps beginners understand:

- Tailwind CSS styling
- JavaScript functions
- DOM manipulation
- Event handling
- Dynamic UI rendering
